TverskyFocalLoss

 TverskyFocalLoss (include_background:bool=True, to_onehot_y:bool=False,
                   sigmoid:bool=False, softmax:bool=False,
                   reduction:str='mean', gamma:float=2, alpha=0.5,
                   beta=0.99)

Compute both Dice loss and Focal Loss, and return the weighted sum of these two losses. The details of Dice loss is shown in monai.losses.DiceLoss. The details of Focal Loss is shown in monai.losses.FocalLoss.

Type Default Details
include_background bool True
to_onehot_y bool False
sigmoid bool False
softmax bool False
reduction str mean
gamma float 2
alpha float 0.5 focal_weight: (float, int, torch.Tensor) = None,
lambda_dice: float = 1.0,
lambda_focal: float = 1.0,
beta float 0.99
Returns None
