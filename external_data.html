download_ixi_data

 download_ixi_data (path:(<class'str'>,<class'pathlib.Path'>)='../data')

Download T1 scans and demographic information from the IXI dataset, then process the demographic information for each subject and save the information as a CSV file. Returns path to the stored CSV file.

Type Default Details
path (<class ‘str’>, <class ‘pathlib.Path’>) ../data Path to the directory where the data will be stored
