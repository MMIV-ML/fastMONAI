pred_to_multiclass_mask

 pred_to_multiclass_mask (pred:torch.Tensor)

Apply Softmax function on the predicted tensor to rescale the values in the range [0, 1] and sum to 1. Then apply argmax to get the indices of the maximum value of all elements in the predicted Tensor. Returns: Predicted mask.

Type Details
pred Tensor [C,W,H,D] activation tensor
Returns Tensor

source

batch_pred_to_multiclass_mask

 batch_pred_to_multiclass_mask (pred:torch.Tensor)

Convert a batch of predicted activation tensors to masks. Returns batch of predicted masks and number of classes.

Type Details
pred Tensor [B, C, W, H, D] batch of activations
Returns (<class ‘torch.Tensor’>, <class ‘int’>)

pred_to_binary_mask

 pred_to_binary_mask (pred)

Apply Sigmoid function that squishes activations into a range between 0 and 1. Then we classify all values greater than or equal to 0.5 to 1, and the values below 0.5 to 0.

Returns predicted binary mask(s).

Type Details
pred [B, C, W, H, D] or [C, W, H, D] activation tensor
Returns Tensor
